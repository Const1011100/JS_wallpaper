Array Object
keys()
values()

Опис: Object.kays повертає масив властивостей із заданого об'єкта.

Синтаксис: Object.kays(obj)
  • obj - об'єкт чиї властивості будуть повернуті.

Опис: Object.values повертає масив значень властивостей із заданого об'єкта.

Синтаксис: Object.values(obj)
  • obj - об'єкт чиї значення властивостей будуть повернуті.
push()
pop()
shift()
unshift()

Опис: push додає елементи в кінець існуючого масиву.

Повертає: push повертає нову довжину масиву після додавання елементів.

Синтаксис: array.push(element1, element2, ..., elementN)
  • array - масив в який додаємо елементи.
  • elementN - елемент чи елементи які додаємо в масив array.

Опис: pop видаляє останній елемент існуючого масиву.

Повертає: pop повертає видаленний елемент.

Синтаксис: array.pop()
  • array - масив з якого видаляємо елементи.

Опис: shift видаляє перший елемент існуючого масиву після чого зсуває всі елементи на одну позицію вліво змінюючи їх індекс.

Повертає: shift повертає видаленний елемент.

Синтаксис: array.shift()
  • array - масив з якого видаляємо елементи.

Опис: unshift додає елементи на початок існуючого масиву.

Повертає: unshift повертає нову довжину масиву після додавання елементів.

Синтаксис: array.unshift(element1, element2, ..., elementN)
  • array - масив в який додаємо елементи.
  • elementN - елемент чи елементи які додаємо в масив array.
